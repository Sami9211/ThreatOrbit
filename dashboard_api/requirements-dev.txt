# Test-only dependencies (NOT installed into the production image).
#
# httpx2 — Starlette >=1.3 prefers `httpx2` for its TestClient and only falls
#   back to the legacy `httpx` shim with a StarletteDeprecationWarning; a future
#   Starlette release drops that fallback entirely. Production code keeps using
#   the stable `httpx` (see requirements.txt); httpx2 is needed only so the test
#   suite runs on the sanctioned client. pytest.ini turns the deprecation into an
#   error so an uninstalled httpx2 fails CI loudly instead of silently
#   regressing to the deprecated path.
#
# Install alongside the base deps when running the tests:
#   pip install -r requirements.txt -r requirements-dev.txt
httpx2>=2.5.0,<3
